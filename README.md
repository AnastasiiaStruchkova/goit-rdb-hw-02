# Домашнє завдання: Проектування баз даних з використанням семантичних моделей

Цей репозиторій містить результати виконання домашнього завдання з нормалізації баз даних та створення ER-діаграм.

## Опис завдання
У процесі виконання було:
1. Проведено нормалізацію вихідної неатомарної таблиці до 1НФ, 2НФ та 3НФ.
2. Розроблено ER-діаграму для нормалізованих таблиць (зв'язки один-до-багатьох).
3. Створено схему бази даних за допомогою MySQL Workbench.

## Структура репозиторію
* `p1_4_normalization.pdf` — звіт з детальним описом процесу нормалізації від нульової до третьої нормальної форми.
* `p5_ER_diagram.jpg` — скриншот розробленої ER-діаграми.
* `p6_workbench_schema.jpg` — скриншот розгорнутої схеми у MySQL Workbench.
* `goit_rdb_hw_02.sql` — SQL-дамп створеної бази даних.

## Технології
* MySQL / MySQL Workbench
* Основи реляційних баз даних
* Нормалізація (1NF, 2NF, 3NF)
